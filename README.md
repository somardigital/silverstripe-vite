# Silverstripe Vite

This module provides the ability to load Vite assets into Silverstripe.

Note: This module does not provide any frontend build tooling

## Installation

```
composer require "somardigital/silverstripe-vite"
```

## Requirements

* php: ^8.1
* silverstripe/cms: ^5
